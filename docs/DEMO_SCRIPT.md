# DEMO SCRIPT（90 秒多场景）

## 0-10s：开场
- 旁白：“链上操作需要策略、风控、执行、支付、复盘，我们把它做成一个副驾驶。”

## 10-35s：场景 1 - 交易版
1. 用户输入：“帮我分3笔买入ETH，总预算1000USDT，30分钟一笔，回撤5%停手。”
2. 画面展示：
   - 意图解析（资产、预算、节奏、风控）
   - 行情快照（OnchainOS 行情能力）
   - 执行步骤：3 笔 TWAP + 风控清单
3. 用户点击“确认执行”。

## 35-60s：场景 2 - 运营版
1. 用户输入：“抓 3 个 Crypto 热点，生成 X 账号可发内容并附交易观察。”
2. 展示：
   - 热点抓取（OnchainOS 数据能力）
   - 自动生成文案 + CTA
   - 交易观察列表 & 风险免责声明

## 60-80s：场景 3 - 支付版
1. 用户输入：“给 0xABC... 打 100 USDT，gas 超 8 USDT 就延后。”
2. 展示：
   - 支付 Quote（Gas、ETA）
   - 合规扫描结果
   - Gas Cap 对比 + 延后策略
   - 用户确认后生成支付回执

## 80-90s：收尾
- 旁白：“Onchain Copilot：一句话，从行情、策略到支付执行和复盘的链上副驾驶。”
