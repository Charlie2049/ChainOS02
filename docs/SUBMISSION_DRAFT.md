## English Version
We built **Onchain Copilot**, an OnchainOS-native AI agent that turns natural-language intents into executable onchain plans with built-in risk controls and post-trade reviews.

The V2 demo highlights three concrete versions:
1. **Trading** – staged execution plans with slippage/position/drawdown checks.
2. **Operations** – pull crypto hot topics → auto-generate X/Twitter copy → attach trading watchlist.
3. **Payments** – pre-trade quote, compliance scan, gas-cap enforcement, optional delay.

The MVP already includes intent parsing, scenario-aware pipelines, and both CLI + Web demos. Next we will plug into real OnchainOS APIs so the same flow can execute on live wallets, markets, and dApps.
