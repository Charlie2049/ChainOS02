最适合落地的 3 个版本

同一套引擎，根据场景切换策略与风控。

交易版

分批交易 + 风控

输入“帮我分 3 笔买入 ETH，回撤 5% 停手”，即可获得行情快照 + 执行步骤 + 滑点/Gas/仓位检查。

  • TWAP / TCA 计划
  • 回撤、滑点、Gas 阈值
  • 复盘摘要与提醒

运营版

热点 + 内容 + 观察

自动抓 Crypto 热点，生成可直接发布的 X 文案，并附带交易观察列表与免责声明。

  • 热点聚合 + 情绪摘要
  • X / 小红书可发模板
  • 交易观察 + 提醒

支付版

链上支付助手

输入“给地址 A 打 100 USDT，Gas 超过阈值延后”，即可获得 Quote、合规扫描、Gas Cap 对比与回执。

  • Gas 费用 + ETA
  • 黑名单/合规检查
  • 支付回执 + 记账

    如何继续？

    当前网站是可运行的 MVP：静态托管即可对外展示，命令行版本也已就绪。

    运行 CLI

    cd onchain-copilot
python3 src/copilot.py --mode trading --text "帮我分3笔买入ETH..."

    部署 Web

    1. `cd onchain-copilot/web`
    2. `python3 -m http.server 8787` 本地预览
    3. 将 `web/` 静态文件上传到 GitHub Pages / Vercel / Netlify / Cloudflare Pages

    接入 OnchainOS

    • 在 `src/copilot/services.py` 替换 mock 为真实 SDK
    • 扩展策略/风控插件，沉淀为可执行工作流
    • 记录执行日志，生成自动复盘
